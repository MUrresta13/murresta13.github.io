How to use
1) Put this index.html at the root of murresta13.github.io (or any folder you want).
2) Place your halloween.mp4 next to it, with that exact name.
3) Open https://murresta13.github.io/index.html?v=1 (add ?v=1 to bypass cache).
Notes
- iPhone requires a tap before audio/video plays; our 'Start the Spook' button provides that.
- If you change the clip length, update VIDEO_LEN_MS in the script.
